## Architecture

- Domain
- Application
- Infrastructure
- API

## Technologies

- .NET 10
- PostgreSQL
- Entity Framework Core
- Clean Architecture

- Upload Biometric Sample
- Store Biographic Data
- Enrollment Workflow
